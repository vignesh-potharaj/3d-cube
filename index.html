Three.js Architecture

Learn the fundamental concepts that render this 3D cube in WebGL.

1. Scene

The THREE.Scene is the container that holds all 3D objects, lights, cameras, and helpers. Think of it as the 3D universe where your geometry resides. Without a scene, Three.js cannot draw anything. 

const scene = new THREE.Scene();
// Objects, meshes, and lights are added to this container
scene.add(cube);
scene.add(pointLight);

2. Camera

The camera determines what part of the scene is displayed and how it is projected. We use a THREE.PerspectiveCamera, which mimics how the human eye views the real world, rendering objects smaller as they move farther away. 

// parameters: FOV (degrees), Aspect Ratio, Near plane, Far plane
const camera = new THREE.PerspectiveCamera(
  45, 
  canvasWidth / canvasHeight, 
  0.1, 
  100
);
camera.position.set(0, 0, 5);

3. Renderer

The THREE.WebGLRenderer uses your computer's GPU via WebGL to draw the 3D scene onto the 2D screen of the browser. It handles resolution, pixel ratio, shadow mapping, and pixel processing. 

const renderer = new THREE.WebGLRenderer({ 
  canvas: document.getElementById('threejs-canvas'),
  antialias: true,
  alpha: true
});
renderer.setSize(width, height);
renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2));

4. Geometry

Geometry defines the physical shape of a 3D object by specifying its vertices (points) and faces. We use a THREE.BoxGeometry to build a perfect cube with specified width, height, and depth dimensions. 

// parameters: width, height, depth
const geometry = new THREE.BoxGeometry(1.5, 1.5, 1.5);
// Generates vertices, normals, and texture coordinates automatically

5. Material

Material defines how the surface of an object reacts to light. We use THREE.MeshStandardMaterial, which uses physically-based rendering (PBR). It responds dynamically to light intensity, metalness, and roughness values. 

const material = new THREE.MeshStandardMaterial({
  color: 0x818cf8,
  roughness: 0.4,
  metalness: 0.2
});

6. Mesh

A THREE.Mesh is a structural object composed of a geometry (shape) and a material (appearance). Once created, we place the mesh at a coordinate in space and add it to our scene to make it visible. 

// Combine geometry and material into a single object
const cube = new THREE.Mesh(geometry, material);
scene.add(cube); // Place it into the scene

7. Lighting

Without lights, standard materials render pitch black. We add a soft, global THREE.AmbientLight for basic visibility, and a directional THREE.PointLight (like a lightbulb) that casts dynamic reflections and specular highlights. 

// Ambient light illuminates all faces equally
const ambientLight = new THREE.AmbientLight(0xffffff, 0.4);
scene.add(ambientLight);

// Point light shines outwards from a specific coordinate
const pointLight = new THREE.PointLight(0xffffff, 2.5);
pointLight.position.set(3, 3, 3);
scene.add(pointLight);

8. Animation Loop

The animation loop is a function called recursively using the browser's native requestAnimationFrame. At each frame, we update the cube's rotation coordinates and call the renderer to draw the updated scene. 

function animate() {
  requestAnimationFrame(animate);
  
  // Rotate the cube slightly on each frame
  cube.rotation.x += 0.01;
  cube.rotation.y += 0.01;
  
  // Render the scene from the camera's perspective
  renderer.render(scene, camera);
}
animate();
